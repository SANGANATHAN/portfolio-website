

## Sanganathan P — Portfolio Website

### Design
- Dark theme with deep navy/black background (#0a0a1a)
- Neon blue/purple glowing accents
- Poppins font for headings, Inter for body
- Animated gradient background with floating particles effect
- Smooth scroll navigation with Framer Motion page transitions

### Structure

**Navbar** — Fixed top nav with links to all sections, hamburger menu on mobile

**Hero Section** — Full-screen with name, role, tagline, profile image placeholder, "Download Resume" & "Contact Me" buttons with glow effects

**About Me** — Card layout with key traits and description

**Education** — Timeline-style card showing B.Tech IT at PSN Engineering College (2023–2027)

**Skills** — Grid of skill cards with icons and hover glow effects (HTML, Data Analytics, Photoshop, Photography, Problem Solving, Teamwork)

**Experience** — Card highlighting Data Analytics value-added course at Infosmite Edutech LLP

**Certifications** — Card with date range for Data Analytics Course

**Achievements** — Card for TECHBLITZ 2025 participation

**Projects** — Placeholder section with "Coming Soon" cards for future use

**Contact** — Contact info (email, phone, address) + contact form UI (name, email, message fields)

**Footer** — GitHub/LinkedIn icon placeholders + copyright

### Animations
- Framer Motion fade-in-up on scroll for each section
- Floating particle/glow background animation
- Hover scale + glow effects on cards and buttons
- Smooth scroll between sections
